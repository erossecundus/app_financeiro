Olá! Esse é um projeto de aplicativo para controle financeiro.

Entre os principais aspectos abordados para a elaboração tivemos o primeiro contato com modelagem e estrutura de banco de dados.
Diagrama conceitual, modelo lógico, entrada de dados e comandos "select", utilizando o MySQL. Também foi explorado os aspectos da criação de protótipos de telas utilizando o figma.
Foi desenvolvido para a disciplina de UPX II, do curso de ADS da Facens em Sorocaba, SP, em 2023.

Apesar do estágio preliminar, sem as integrações, o projeto atendeu às expectativas.
